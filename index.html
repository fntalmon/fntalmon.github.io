Proyecto Cines Frontend

HTML,CSS(Tailwind),JS(React)

App web que permite que usuarios se registren, consulten por las peliculas disponibles y sus funciones y adquieran entradas para una funcion en particular, adquiriendo productos de un candy bar si asi lo desean. Proyecto Cines Backend

PHP(Laravel)

Esta app forma parte de un proyecto realizado para una materia de la universidad. Ademas, implemente una API Rest que es consumida por el front-end, tanto para poblar la pagina como para la compra de entradas. PHP(Laravel)

This app is part of a project carried out for a university subject.
